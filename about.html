🇮🇳 About Desi Jugad

Jugaad for the modern Indian.

Desi Jugad — fast, free, and fiercely private tools for everyone. We got tired of converter sites that make you sign up just to resize a photo or merge a PDF. So we built this. No account. No upload. Seedha kaam ho gaya.

We're an independent team in India. We started with one simple idea: modern browsers are powerful enough to do the conversion themselves — why upload your personal files to a random server?

🔒 Privacy ka promise

Jab aap koi file Desi Jugad pe drop karte ho, woh file sirf tumhare device pe rehti hai. Conversion tumhare browser ke memory mein hoti hai — JavaScript aur WebAssembly ke through. Koi server nahi. Koi upload nahi. Koi data collection nahi.

Agar yakeen na ho — apna browser ka Network tab kholo aur dekho. Tumhara file kahin nahi jayega.

🇮🇳 Made in India, built for the world

Hum India se hain — team, server, taxes — sab yahan. We believe great internet tools should come from everywhere, not just Silicon Valley. A student in Bengaluru, a shopkeeper in Lucknow, a designer in Lagos — sabke liye same experience. Free, fast, private.

🎁 Hamesha free kaise?

Koi subscription nahi. Koi pro tier nahi. Site ko unobtrusive display ads support karte hain. Aur kyunki hamare paas file process karne ke liye koi server nahi hai, running cost genuinely kam hai — domain aur CDN ke liye bas thoda paisa.

Humne apne aap se promise kiya hai: dark patterns nahi, sign-up gates nahi, aur site kabhi kisi ko nahi bechenge. Agar kabhi majboori hoi, toh loudly batayenge — koi chhupa chhupa nahi.

🛠️ Open source, open libraries

Hamare saare conversion libraries open source hain — GitHub pe milte hain, bade companies use karte hain, thousands of developers review karte hain. Hum use karte hain heic2any, pdf.js, jsPDF, SheetJS, js-yaml, marked.js aur aur bhi. Desi Jugad sirf ek aasaan interface hai — jugaad at its best.
