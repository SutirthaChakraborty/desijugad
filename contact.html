Before you email — common questions

Can you process my file for me?

We genuinely can't. Our tools run in your browser, and we never receive your files on any server. If you're having trouble converting a specific file, paste the error message into an email and we'll help you debug.

Do you offer an API?

Not at the moment. All our conversion logic uses open-source JavaScript libraries that you can use directly in your own project — we'd rather point you to those than lock you into our API. If you'd find a hosted API useful, let us know; enough interest and we'll consider it.

Can I white-label Desi Jugad for my company?

The site is built as a clean static project. If you're a company that wants a privacy-first converter inside your own product, write to us at hello@desijugad.co.in and we'll see if we can help.

Why haven't I received a reply?

Check your spam folder first — replies from small domains often land there. If it's been more than a week, resend your message and we'll dig it up. We do not use automated ticketing; every reply is written by a human, so busy weeks can stretch the queue.

I found a security issue. How do I report it?

Please email security@desijugad.co.in directly with details. We take security seriously and will acknowledge your report within 48 hours. Since our tools run entirely client-side, the surface area is small, but we still welcome responsible disclosure.

Stay in the loop

We're planning to add a tiny, infrequent newsletter — one email a month at most, only when we ship new converters. No tracking pixels, no marketing. If that sounds good, email us with "subscribe" in the subject line and we'll add you to the list.
