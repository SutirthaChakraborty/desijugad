Which favicon sizes do I need?

  • 16×16 — Browser tab favicon, bookmarks bar
  • 32×32 — Browser tab on high-DPI screens, taskbar
  • 48×48 — Windows site icon
  • 180×180 — Apple Touch Icon (iOS home screen)
  • 512×512 — Progressive Web App (PWA) icon, Android splash
