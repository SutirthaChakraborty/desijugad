PDF merging and splitting, explained

This tool uses pdf-lib, a pure-JavaScript PDF library that runs entirely in your browser. No server processing, no file upload, no size limits beyond your device RAM.

Merge

Drop multiple PDFs and they'll be concatenated in order. You can drag the items in the list to reorder them before merging. Embedded fonts, images, and forms are all preserved.

Split / Extract

Choose a specific page range (e.g. 1,3,5-8) to extract only those pages into a new PDF, or download each page as a separate file in a ZIP.

FAQ

Is there a page limit for merging?

No hard limit — only your browser's memory. We've successfully merged 200-page PDFs on a standard laptop.

Are encrypted/password-protected PDFs supported?

Not currently. Password-protected PDFs need to be unlocked first. This is a browser security restriction.

Will annotations and form fields be preserved?

Yes — pdf-lib copies all page content including annotations. Interactive form fields are preserved in the output PDF.
