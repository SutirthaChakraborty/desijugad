Convert between any image format, in a second

This tool handles conversion between all the image formats that modern browsers natively understand: PNG, JPG (JPEG), WebP, and BMP. It's powered by the browser's own Canvas API — the same technology that renders images in every web page you visit — so conversions are fast, accurate, and completely private.

Which format should you choose?

  • JPG — best for photographs where file size matters. Universally supported, slightly lossy.
  • PNG — best for screenshots, logos, illustrations, and anything with transparent backgrounds. Lossless, larger files.
  • WebP — modern format with 25–35% smaller files than JPG at the same quality. Best for websites.
  • BMP — uncompressed, enormous file sizes. Use only when a specific tool demands it.

Batch conversion and resizing

You can drop as many images as your browser can hold in memory. If you set a max width, every image taller or wider than that will be scaled down proportionally — perfect for prepping photos for email, WhatsApp, or a website without ever opening Photoshop.

FAQ

Does conversion reduce image quality?

If you convert to JPG or WebP, there is some lossy compression — controllable via the quality slider. Converting to or from PNG is lossless. The tool defaults to a quality of 0.92, which is visually indistinguishable from the original for photographs.

Can I convert 100+ images at once?

Yes — the only bottleneck is your device's memory. We've tested batches of 200+ images on a modest laptop without issue.

Will transparency be preserved?

Converting to PNG or WebP preserves transparent backgrounds. Converting to JPG or BMP replaces transparency with white, because those formats don't support alpha channels.
