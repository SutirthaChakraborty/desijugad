About image flipping

This tool uses the Canvas API with ctx.scale(-1, 1) for horizontal flipping and ctx.scale(1, -1) for vertical flipping. The transform is applied relative to the canvas center, so the output image has the same pixel dimensions as the input.

Common uses for flipping images

  • Selfie mirror effect — Front cameras on phones often produce a mirrored image. Flip horizontally to correct it.
  • Symmetry art — Create kaleidoscope effects or symmetric patterns by flipping and overlaying.
  • Text in images — Flip horizontally to correct backwards text from certain scans or screenshots.
  • Print layouts — Some print templates require a mirrored image for iron-on transfers.

FAQ

Does horizontal flip and mirror mean the same thing?

Yes — a horizontal flip creates a mirror image, reflecting the image along the vertical axis. The left side becomes the right side, like looking in a mirror.

Does flipping an image reduce quality?

No quality loss occurs during a flip operation since it is a geometric transform only. For JPG output, the quality slider controls the re-encoding quality. Use PNG or WebP for truly lossless output.

Can I flip an animated GIF?

This tool currently supports PNG, JPG, and WebP. Animated GIF flipping is not supported — the first frame would be processed as a static image.
