One converter for all spreadsheet formats

Different tools create different formats. Excel creates XLSX (and the older XLS), LibreOffice Calc uses ODS, databases export CSV. This tool bridges all of them using SheetJS — the most widely used spreadsheet parsing library on the web.

Supported formats

  • XLSX — Modern Excel format (Office Open XML), supported by Excel 2007+, Google Sheets, LibreOffice
  • XLS — Legacy binary Excel format. All versions of Excel, older tools
  • CSV — Universal plain text. Works in everything
  • ODS — Open Document Spreadsheet. LibreOffice, OpenOffice
  • TSV — Tab-separated values. Common in bioinformatics and data science
  • JSON — Array of row objects. Perfect for developers
  • HTML table — Paste directly into any webpage

FAQ

Will formulas be preserved?

When converting to XLSX, formulas are preserved. When converting to CSV, TSV, or JSON, only the computed values are used — formulas become their results.

How many rows can this handle?

SheetJS handles hundreds of thousands of rows comfortably. For very large files (50 MB+), parsing may take a few seconds. Everything runs in your browser's memory.

Can I select a specific sheet from a multi-sheet workbook?

Yes — use the Sheet dropdown to switch between sheets. The preview and download will reflect the selected sheet.

What happens to merged cells?

Merged cells in Excel may not translate perfectly to CSV or TSV. The top-left cell of the merge retains the value; other cells will be empty.
