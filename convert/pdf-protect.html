How PDF password protection works

This tool uses pdf-lib to apply AES/RC4 encryption to your PDF in-browser. Two types of password exist in the PDF specification:

User password

Anyone who wants to open the PDF must enter this password. Without it, the file cannot be viewed. This is the most common protection for confidential documents.

Owner password

This password controls the permissions of the PDF — printing, copying, editing. Someone who knows the owner password can override restrictions. If only an owner password is set, the file can still be opened by anyone but actions are restricted.

Permissions

Uncheck "Allow printing" to prevent low-res printing. Uncheck "Allow copying text" to block text selection and clipboard copy. These restrictions are enforced by compliant PDF readers.

FAQ

Does this work on mobile?

Yes — works on Chrome, Safari, and Firefox on iOS and Android. The protected PDF downloads to your device just like on desktop.

Can I protect a PDF that's already password-protected?

Not directly. Unlock it first with our PDF Unlock tool, then re-protect with new passwords.

What happens if I forget my password?

There is no recovery option. Write down or save your password in a password manager before protecting the file.
