FAQ

Can I screenshot a live website URL?

Browser security prevents capturing content from other domains directly. For live URL screenshots, use browser extensions like GoFullPage or FireShot, or online services like screenshotmachine.com. This tool works best with HTML you paste directly.

What is the difference between PNG and JPG screenshots?

PNG is lossless — perfect for text, UI mockups, and anything with sharp edges or flat colors. JPG is lossy but produces smaller files — better for photographs and design previews where pixel-perfect accuracy isn't required.

Why does the screenshot look different from the live page?

html2canvas re-renders the DOM using a Canvas API approximation. Some CSS features (certain gradients, blend modes, SVG filters, web fonts from external CDNs) may render differently than a real browser screenshot.

How do I take a high-resolution screenshot for print?

Use 3× scale for print-quality output at 216 dpi. For very high resolution (300+ dpi), use a dedicated tool or set your browser zoom level to 200% before using the browser's Capture full size screenshot DevTools option.
