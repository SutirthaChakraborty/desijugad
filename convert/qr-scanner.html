FAQ

Does this work on mobile?

Yes — tap "Start Camera" and it will request camera permission and scan in real-time.

Why does camera scanning need HTTPS?

The browser's getUserMedia API requires a secure context (HTTPS) to access the camera. This site uses HTTPS so everything works fine.
