How PDF cropping works

PDF cropping sets a crop box on each page, which tells PDF viewers which portion of the page to display. Content outside the crop box is hidden but not deleted — this is standard PDF behaviour.

Setting crop margins

Enter how much to trim from each side in millimetres. The live diagram shows the resulting crop area. The tool applies the same crop to all pages. 1 mm = 2.83465 PDF points.

Use cases

Remove oversized white borders from scanned documents. Crop header/footer regions from academic papers. Trim binding area from photographed book pages. Remove repeated watermarks placed in margins.

FAQ

Can I crop different amounts from different pages?

This tool applies uniform crop values to all pages. For per-page cropping, use PDF Split to separate pages, crop individually, then merge them back.

What happens if I crop too much and cut into the content?

The crop box simply hides more of the page. You can re-open the original file and try again with smaller values — the source file is never modified.

Does the crop affect printing?

Yes. PDF viewers and printers typically honour the crop box, so only the visible (cropped) area is printed. Some professional print workflows may use the media box instead.
