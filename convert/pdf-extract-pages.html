Extracting pages from a PDF

This tool creates a new PDF containing only the pages you specify. The original file is unchanged. Useful for extracting a chapter, a certificate, a diagram, or any subset of a larger document.

Page range syntax

Use 1,3,5-8 to get pages 1, 3, 5, 6, 7, 8 in that order. Ranges are inclusive and 1-based. You can mix individual numbers and ranges freely.

Preserving quality

Extracted pages are byte-for-byte copies of the originals — no re-encoding, no quality loss. Fonts, images, and embedded metadata are all intact.

FAQ

Can I extract pages in a different order than the original?

Yes. Enter pages in whatever order you want: 5,3,1 gives you pages 5, 3, 1 in that sequence in the output PDF.

Can I extract pages from a scanned PDF?

Yes. Scanned PDFs (image-based) work exactly the same as text PDFs. Each page is copied as-is.

What is the difference between this and the Split tool?

Extract puts all selected pages into one new PDF. Split creates multiple PDFs — one per range or chunk. Use this tool when you want a single output file from specific pages.
