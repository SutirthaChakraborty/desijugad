Precision image resizing — width, height, percentage, or DPI

This tool gives you full control over output image dimensions. Choose your resize mode and set exact target values. Everything runs inside your browser using the Canvas API — no uploads, no servers, no data leaves your device.

Resize modes explained

  • Exact W × H — Set precise pixel dimensions. Toggle "Maintain aspect ratio" to avoid stretching.
  • Max width / Max height — Constrain one dimension; the other scales proportionally.
  • Percentage — Scale to 50%, 200%, or any factor relative to the original.
  • Target DPI — Useful for print preparation. E.g., if your image is 72 dpi and you need 300 dpi for a brochure, this recalculates pixel dimensions accordingly.
  • Fit inside box — Fit image into a W × H bounding box, preserving aspect ratio.

What is DPI and why does it matter?

DPI (dots per inch) determines how sharp an image appears when printed. A standard photograph needs 300 DPI for quality prints. Web images are typically 72–96 DPI. Use the "Target DPI" mode to calculate the pixel dimensions needed for a desired print resolution.

FAQ

Can I resize while keeping the same format?

Yes — set Output format to "Same as input" and the file extension will be preserved.

Can I upscale images?

Yes, using Percentage mode (e.g. 200%) or exact W × H values larger than the original. Note that upscaling cannot add detail — it simply enlarges existing pixels.

Is there a maximum file size?

Only your device's memory is the limit. We've tested batches of 100+ multi-megapixel images without issues on a standard laptop.
