How PDF unlocking works

This tool uses pdf-lib to load your PDF with the password you provide, then saves a new copy without any password or permission restrictions. The process is 100% in-browser — your file and password are never transmitted anywhere.

What this tool removes

It removes both the open (user) password and any permission restrictions (printing, copying, editing). The output PDF is fully accessible without any password.

What this tool cannot do

This is not a password cracker or brute-force tool. You must already know the correct password. If you do not have the password, this tool cannot help you.

FAQ

My PDF has restrictions but no open password — can I still unlock it?

Yes. Some PDFs restrict printing or copying with an owner password but don't require a password to open. In this case, leave the password field blank (or try an empty string) and click Unlock — pdf-lib can often clear owner-only restrictions.

Will the unlocked PDF look different?

No. The content, layout, fonts, and images are preserved exactly. Only the encryption and password protection is removed.

Can I re-protect the PDF after unlocking?

Yes. Use our Protect PDF tool to add a new password after unlocking.
