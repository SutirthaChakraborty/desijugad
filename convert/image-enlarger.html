How image upscaling works

This tool uses the browser's Canvas API with imageSmoothingQuality = 'high', which applies bicubic-style interpolation when drawing an image at a larger size. Each new pixel's value is computed from surrounding pixels, producing smooth gradients and edges.

Upscaling tips

  • Start with the highest quality source — Upscaling cannot create detail that was never there. Start with the best quality original.
  • 2x is the sweet spot — For most images, a 2x upscale with bicubic interpolation is virtually indistinguishable from the original at full size.
  • Use PNG output — For lossless upscaling results, select PNG or WebP as the output format.
  • Large images take more time — A 4000×3000px image upscaled 4x creates a 16000×12000px canvas which is demanding on browser memory.

FAQ

What is bicubic interpolation?

Bicubic interpolation calculates the value of each new pixel by looking at the 4×4 grid of nearest source pixels and computing a weighted average. This produces smoother edges than bilinear (2×2 grid) or nearest-neighbor (1 pixel) methods. The browser's Canvas API uses this when imageSmoothingQuality is set to 'high'.

Why does my upscaled image look blurry?

Interpolation-based upscaling applies smoothing which can appear soft at very high scale factors (4x+). For sharper results at large scales, use AI upscaling tools. This tool is best for 1.5x–2x enlargements.

Are my images uploaded to a server?

No. Everything runs in your browser. Your images are never sent to any server.
