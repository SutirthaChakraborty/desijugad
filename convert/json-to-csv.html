Why convert between JSON and CSV?

JSON is the web's favorite data format — hierarchical, expressive, and perfect for APIs. CSV is the universal spreadsheet format — flat, familiar, and openable by every data tool on earth. Moving between them is one of the most common data tasks, and this tool does it locally with nothing uploaded.

Nested objects, handled

When you convert JSON with nested structure to CSV, there's no single "right" answer — CSV doesn't have nesting. By default we flatten nested keys using dot notation (so user.profile.name becomes a column). Toggle the option to instead stringify nested values as JSON strings.

FAQ

Does this work for huge files?

Up to a few million rows, yes. Everything runs in memory on your device.

Will my data be sent anywhere?

No. This is a pure JavaScript parser. Check your browser's Network tab — not a single byte leaves your device during conversion.

What about CSV quirks like quotes and commas inside fields?

Values containing delimiters, quotes, or newlines are automatically quoted and escaped per RFC 4180.
