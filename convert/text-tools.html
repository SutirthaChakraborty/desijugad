Quick reference

Case conversions — handy for CSS classes, variable names, titles, or normalizing database fields. Title Case capitalizes each word; Sentence case capitalizes only the first word.

URL encode / decode — for when you need to safely put text in a URL query string, or read one back.

HTML escape / unescape — turn <, >, &, " into their entity equivalents, so they display literally instead of being parsed as HTML.

Cryptographic hashes (SHA-1, SHA-256, etc.) — produce fixed-length fingerprints of input. Useful for checksums, cache keys, and verifying file integrity. Note that SHA-1 should not be used for security-critical applications anymore.

Slugify — turn "Hello, World!" into hello-world for use in URLs and file names.

FAQ

Are the hashes calculated locally?

Yes — we use the browser's native SubtleCrypto API. No text is ever sent over the network.

Why no MD5?

MD5 is broken for security purposes and not part of the browser's SubtleCrypto. If you need MD5 for legacy reasons, it's a few-line addition to this page — but we don't recommend it for anything important.

Can I chain operations?

Run one, copy the result, paste back into the input, run the next. Full chaining is on our roadmap.
