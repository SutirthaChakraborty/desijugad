How video-to-GIF conversion works

FFmpeg uses a two-pass approach for high-quality GIFs: first it generates an optimal palette from the video frames, then it converts using that palette. This produces much better color accuracy than naive conversions.

FAQ

Why are GIF files so large?

GIF uses LZW compression which is far less efficient than modern video codecs. A 5-second 480px GIF at 10fps can easily be 3–8 MB. For web embedding, consider WebP animation or a short MP4 loop instead.

How do I make a looping GIF?

All GIFs generated by this tool loop infinitely by default, which is standard GIF behavior.
