How EPUB works

An EPUB file is actually a ZIP archive containing HTML/XHTML files (one per chapter), a CSS stylesheet, images, and an OPF metadata file that lists the reading order. This tool unzips the EPUB, reads the OPF to find the correct chapter order, extracts clean text from each HTML chapter, and uses jsPDF to write it to PDF pages.

Why not use Calibre?

Calibre handles EPUB→PDF with full layout fidelity, custom fonts, and images. This browser tool is for quick conversions when you don't want to install software. It's instant, private, and requires no account.

FAQ

Does this handle DRM-protected EPUBs?

No. DRM (Digital Rights Management) encrypted EPUB files cannot be opened by this tool. You would need to remove DRM first (which may be restricted by your jurisdiction's copyright laws).

What about EPUB 3 files?

Both EPUB 2 and EPUB 3 are supported for text extraction. EPUB 3 media overlays (audio sync), scripted content, and fixed-layout features won't transfer to PDF.

Can I convert multiple EPUBs at once?

Currently one file at a time. For batch conversion, Calibre's command-line interface is ideal.

The PDF is missing some content from my EPUB

Some EPUB files use JavaScript for interactive content or have unusual packaging. This tool extracts static HTML content. If chapters are missing, the EPUB may use non-standard spine ordering or encrypted content.
