ZIP files — the universal archive

ZIP is the world's most common archive format. Every operating system can open it natively — Windows, macOS, Linux, Android, iOS. It combines multiple files into one, with optional compression, making sharing and storage easier.

CREATE mode

Drag any number of files onto the zone. The tool uses JSZip to bundle them into a single .zip archive in memory, then triggers a download. No data leaves your device.

EXTRACT mode

Drop a .zip file and the tool reads its directory listing. Click any filename to download that individual file, or use "Download all" to save each file one by one.

FAQ

Can I add folders to the ZIP?

You can select files from inside folders. To add entire folder structures, use the Create Archive tool which supports webkitdirectory folder picking.

Does extraction support password-protected ZIPs?

Password-protected ZIPs are not currently supported — JSZip's password support is limited. Use a desktop tool like 7-Zip for encrypted archives.

What compression level is used?

JSZip uses DEFLATE compression at level 6 (the standard default). For files that are already compressed (JPG, MP4, MP3), the size reduction will be minimal.

Can I extract RAR or 7z files here?

This tool only supports ZIP format. For RAR and 7z, try a desktop app like 7-Zip (free and open-source). CBZ files (comic books) are ZIP files and will work fine here.
