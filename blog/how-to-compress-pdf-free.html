How to compress PDF file size free without losing quality (2026)

8 min read·April 2026

Your PDF is 15MB and the email portal only allows 5MB. Or you need to upload a scanned document to a government portal that has a strict file size limit. Or you're storing thousands of PDFs in Drive and the storage costs are adding up. Compressing PDFs solves all of these — and you can do it free in your browser without uploading files to any server.

Why PDFs get so large

A PDF file is a structured container format. What's inside it determines the size:

  • Embedded images: The #1 cause of large PDFs. A scanned document at 300 DPI with high-resolution images embedded can easily reach 10–20MB per page.
  • Uncompressed fonts: PDFs embed font files. Multiple fonts or full Unicode font sets add megabytes.
  • Duplicate resources: Some PDF creators (Word, Illustrator) embed duplicate objects — the same image referenced multiple times stores multiple copies.
  • Hidden data: Revision history, hidden layers, comments, and form data accumulate over time.
  • Unoptimised export settings: Exporting a Word document or InDesign file to PDF with "Press Quality" or "High Resolution" settings when you only need screen quality.

How PDF compression actually works

There are two main approaches to reducing PDF size:

1. Image recompression

Most PDF compression tools work by re-encoding the embedded JPEG images at a lower quality. This is what our browser-based tool does using pdf-lib + Canvas API. If a scanned page has a 2MB embedded JPEG, re-encoding it at 65% quality might bring it down to 500KB — a 75% size reduction for that page. Text and vector graphics are untouched.

2. Structure optimisation

Professional tools like Ghostscript or Adobe Acrobat can also optimise the PDF structure — removing duplicate objects, compressing fonts, cleaning revision history, and rebuilding the cross-reference table. This can reduce file size by 10–30% on top of image compression.

When compression has no effect

If the PDF consists entirely of text and vector graphics (no embedded raster images), our tool and most browser-based compressors will have minimal effect — because there are no JPEG images to re-encode. Examples: a contract drafted in Word with no images, a spreadsheet exported to PDF, a code document.

For these documents, Ghostscript's screen or ebook output intents can sometimes still reduce size by 10–20% through structural optimisation.

Common use cases in India

  • Income tax portal (incometax.gov.in): Has a 5MB limit for most documents. Scanned ITR forms often come in at 8–15MB.
  • IRCTC train booking: Documents for Tatkal bookings require ID proofs under 100KB.
  • Bank KYC forms: Statement PDFs need to be under 2MB.
  • College admissions: Mark sheets and certificates typically must be under 500KB.
  • WhatsApp and email: PDFs above ~10MB are rejected or delayed.
