Free AI image generation that actually runs on a static site

This generator uses Pollinations.ai as the default provider — it's a free, open project that returns generated images directly as URLs. No key. No sign-up. No watermark. The request goes from your browser to Pollinations and back. We never see your prompt.

For higher quality, switch to Gemini and paste your free Google AI Studio key in Settings. Gemini's API allows browser-direct calls, so your key stays in your browser and goes only to Google. Imagen 3 produces stronger photorealism, more accurate text rendering, and better prompt adherence than free models.

Why we removed Replicate, Suno and OpenAI

None of those providers allow direct browser API calls — they require a server-side proxy, which contradicts our static-only promise. Read more on the AI Studio overview.

Tips for better prompts

  • Be specific: subject, environment, lighting, lens. "A woman in a blue saree" is weaker than "a young woman in an indigo Banarasi silk saree, golden hour, shallow depth of field, 50mm lens".
  • Add style anchors: "photorealistic", "watercolor", "isometric 3D", "low-poly", "studio Ghibli".
  • Use the size preset that matches your end use — Story 9:16 for Instagram, YouTube 16:9 for thumbnails.
  • If a prompt looks wrong, try a new seed before changing the prompt.
