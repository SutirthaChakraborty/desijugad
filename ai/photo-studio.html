Edit any photo with a sentence

The Photo Studio uses Gemini's multimodal capability: upload an image, describe the change you want in plain English, and Gemini returns a new image. Common uses: passport-photo backgrounds, LinkedIn-style headshots, cinematic colour grades, removing objects from a scene, restoring old family photos.

Privacy

Your photo is sent in a single API call to Google's Gemini service using your own key. We don't see it. We don't keep it. Google's data-handling policy applies to that call (see Google's API terms).

What if Gemini refuses?

Gemini's safety filters block requests involving real people in compromising scenarios, watermark removal, deepfakes of public figures, and similar. If a request is refused, the tool shows the refusal reason. Try a different framing, or use the offline Photo Editor for manual edits.

Limits

  • Output resolution depends on the model — usually 1024 px on the long side.
  • The original Replicate-backed face restoration / lighting flows are gone — see why on the AI Studio overview.
