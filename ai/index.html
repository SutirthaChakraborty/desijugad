What changed from the old AI Studio?

We removed Replicate, Suno, OpenAI, and Anthropic integrations because none of them allow browser-direct API calls — even with the user's own key. Browser security (CORS) blocks those calls; the only way to use them was via a server-side proxy, which contradicts our "100% static, no backend" promise. The honest fix was to keep only the providers that work without a server: Google Gemini (BYOK), Pollinations.ai (free, no key), Hugging Face (optional BYOK), and on-device ML for background removal.

What we lost

FLUX, Veo, Kling, Seedance, Wan, Suno V5.5, Claude/Llama via Replicate. These are excellent models — they just can't run from a static site without a paid backend.

What we kept (and what's new)

High-quality image generation (Gemini Imagen + Pollinations FLUX), strong long-form writing (Gemini 2.5), solid multimodal photo editing (Gemini), and a fully offline background remover that no commercial competitor matches for privacy.

Pricing reality check

Pollinations: free. Gemini: free tier covers most casual use, paid tier is roughly ₹0.05–₹2 per image depending on the model. Hugging Face: free tier with rate limits, paid tier scales with usage. You always pay the provider directly — Desi Jugad takes zero margin.
